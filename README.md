# SafeDoc Injection Detector

Uma ferramenta de segurança (AI Safety) de código aberto para inspeção e sanitização de documentos, projetada para prevenir ataques de *Indirect Prompt Injection* antes do processamento por Inteligência Artificial.

## Tecnologias Utilizadas
- **Backend:** Go (Golang)
- **Motor de Análise:** Gemini API

## Como executar
1. Inicialize o módulo: `go mod init safedoc-backend`
2. Execute a aplicação: `go run main.go`
